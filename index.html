Clase 3

Comando para Git

  1. Primer comando: git init (para iniciar un git, una sola vez por proyecto)
  2. Segudo comando: git status (para chequear un estado de git en nuestro proyecto)
  3. tercer comando: git add . (para traquear todos los archivos)
  4. Cuarto comnado: git commit -m "mensaje" (para hacer el paquetito de una version,que podemos subir a la nube)
